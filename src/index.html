This page is only meant to serve as a barebones way of sharing my projects. It is not an exercise in web design.

Web Development Projects

OpenLayers with Download Widget

Simple GIS Web Application built using OpenLayers. The map contains two different layers which are both hosted on my own GeoServer. The layer download widget allows the user to download both layers in a variety of formats. It works by taking advantage of the WFS specification which allows specifying an output format when querying the feature service.

Svelte To Do List

The classic to do list app built using SvelteKit and Tailwind CSS. Svelte is my new favorite framework. I love its simplicity, the fact that it compiles to vanilla JavaScript, and its raw speed.
